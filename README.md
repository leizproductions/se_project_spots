# Spots

A simple, beginner-friendly image-sharing webpage built as part of my TripleTen software engineering program. This project focuses on practicing responsive layout, working with design specs, and building clean, organized HTML and CSS.

## 📌 Project Overview

Spots is a small image-sharing app interface where users can view a profile section and a gallery of image cards. The goal of this project was to learn how to:

- Work with Figma design specs
- Build responsive layouts using CSS Grid and Flexbox
- Organize a project using the BEM methodology
- Use custom fonts with `@font-face`
- Create clean, readable, beginner-friendly code

This is one of the early foundational projects in the program, helping build the skills needed for more advanced interactivity later on.

## ✨ Features

- Responsive layout that adapts to different screen sizes
- Profile section with editable user information
- Image grid built with CSS Grid
- Custom fonts using `@font-face`
- Clean, structured CSS using BEM

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Flexbox & Grid  
- BEM Methodology  
- Custom Fonts (Poppins)

## 📚 What I Learned

- How to read and follow design specs from Figma
- How to structure a webpage using semantic HTML
- How to build responsive layouts using media queries
- How to load and use custom fonts
- How to keep CSS organized using BEM
- How to debug layout issues and match a design closely
